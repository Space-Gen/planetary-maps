## üõ† Mars Data Specifications
- **Terrain**: MGS MOLA Global 463m Digital Elevation Model (DEM).
- **Imagery**: Viking MDIM 2.1 Colorized Global Mosaic (232m).
- **Ellipsoid**: Custom Mars Ellipsoid implementation (3396.19 km radius).
- **Features**: IAU-verified Martian nomenclature including Olympus Mons and Valles Marineris.
